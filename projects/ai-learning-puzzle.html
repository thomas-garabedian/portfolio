Projet académique autour d’un puzzle éducatif, centré sur la modélisation mathématique puis l’implémentation en Java.

Academic project around an educational puzzle, focused on mathematical modeling and Java implementation.

ContexteContext

Projet académique mené de janvier à juin 2025 autour d’un puzzle éducatif. Le travail a été réalisé avant que le « vibe-coding » ne devienne une pratique courante dans nos projets : il reposait donc davantage sur la formalisation du problème, la conception d’un modèle mathématique et sa traduction en code Java.

Academic project carried out from January to June 2025 around an educational puzzle. The work was done before “vibe-coding” became a common practice in our projects: it therefore relied more on problem formalization, mathematical modeling and translation into Java code.

Ce que j’ai faitWhat I did

  • Réflexion principale sur le modèle mathématique du puzzle : états, mouvements autorisés, contraintes et conditions de résolution.
  • Formalisation d’une logique exploitable ensuite dans le code, plutôt qu’un simple assemblage d’interface.
  • Implémentation en Java d’une base logicielle permettant de représenter les interactions, les essais et la progression.
  • Mainly worked on the mathematical model of the puzzle: states, allowed moves, constraints and solving conditions.
  • Formalized a logic that could then be implemented in code, rather than simply assembling an interface.
  • Implemented a Java software base to represent interactions, attempts and progression.

ApprocheApproach

L’intérêt du projet était de passer d’un problème concret à une représentation formelle : identifier les variables pertinentes, définir les transitions possibles, puis traduire cette structure dans une logique de programmation Java.

The value of the project was to move from a concrete problem to a formal representation: identify relevant variables, define possible transitions, then translate this structure into Java programming logic.
